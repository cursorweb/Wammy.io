# Wammy.io
Wammy.io is a fun game where tanks shoot at each other.

Like diep.io and arras.io, but with opinions from me.

## How to play
- WASD/Arrow Keys to move.
- Click/E to shoot
- Right click to use special ability (if any)
- ... more coming soon!

## Development
I am using webpack.

**Develop** - auto-build on changes.
```sh
npm run dev
```

**Production**
```sh
npm run build
```
